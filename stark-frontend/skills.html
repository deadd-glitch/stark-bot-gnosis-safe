About Skills

Skills are defined in SKILL.md files with YAML frontmatter containing metadata and a prompt template that gets injected into the AI's context. 

---
name: code-review
description: Review code for issues
version: 1.0.0
requires_tools: [read_file]
arguments:
  language: { default: "any" }
---
You are a code reviewer. Review the code...

Skill Sources (priority order):

  • Managed - ./skills/managed/
  • Bundled - ./skills/bundled/ (lowest priority)
